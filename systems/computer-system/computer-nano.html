Specs

  • Micro-USB Power: 5V 2.5 Amps (Current Configuration)
  • Barrel Jack Power: 5V 4 Amps Setup
  • Max Est. Power Consumption: 12.5 Watts
  • M.2 Slot: 802.11ac WiFi/BT adapter
  • Storage: 32 GB FashDisk

Software

  • OS: Ubuntu 20.04
    • https://qengineering.eu/install-ubuntu-20.04-on-jetson-nano.html ^links
  • ROS2 
ssh jetson@192.168.1.14

Caveats

  • The Jetson Nano uses a custom kernel (GNU/Linux 4.9.253-tegra aarch64) that can't properly install WireGuard due to a WireGuard bug. The workaround is to use a user space implementation, wireguard-go
